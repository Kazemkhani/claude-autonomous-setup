Wiring diagram
4 nodes · full loop · $0/mo extra
INPUT
→
AirPods → Whisper.cpp (local)
Voice captured via mic. Whisper.cpp transcribes on-device — no cloud, no latency, no privacy leak. Raw text out.
// open-source · runs on M-series · <200ms
ROUTING
→
Telegram bot → Claude API
Transcribed text hits a Telegram bot. Bot forwards to Claude Code via API with system prompt + context. Stateless per message.
// telegram free tier · claude credits
OUTPUT
→
Claude Code → git push / file edit
Claude executes the instruction: writes code, edits files, drafts commit messages, runs shell commands. Results logged to stdout.
// claude-code cli · bash tool surface
FEEDBACK
→
Telegram reply → macOS say (TTS)
Claude's response sent back via Telegram. macOS `say` reads it aloud to AirPods. Full loop — eyes-free, hands-free.
// macos built-in · zero extra install
